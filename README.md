### 프로젝트 소개  
------------------------------------

점차 증가하는 채식에 대한 관심에  어플로, 위치 기반 추천 시스템을 통해 쉽고 빠르게 채식을 실천할 수 있습니다.  
  1. 채식의 단계 선택
  2. 현 위치를 기반으로 가까운 채식 식당 추천
  3. 식당 클릭 시 메뉴, 가격, 오픈 시간, 브레이크 타임, 평점 등 제공
  4. 오늘 자신이 실천한 채식 한 끼가 환경에 미친 긍정적인 영향을 시각화
